# Audit Scope

## Audit Objectives

The primary objective of this security audit is to evaluate the current cybersecurity posture of VORTEX SECURE by identifying security risks, vulnerabilities, and compliance gaps across the organization's critical assets.

The assessment aims to provide actionable recommendations that strengthen security controls, reduce cyber risk, improve regulatory compliance, and increase the organization's overall security maturity.

---

## Scope of Assessment

This security assessment covers VORTEX SECURE's enterprise IT environment, including corporate infrastructure, cloud services, security operations, AI engineering systems, web applications, and internal network architecture.

The audit includes identity and access management systems, Microsoft Azure cloud infrastructure, Active Directory, Microsoft 365, GitHub Enterprise repositories, SIEM platform, VPN services, web applications, and security monitoring capabilities.

Both internal and external attack surfaces are included within the scope of this assessment.

---

## In-Scope Assets

- Customer Database
- Employee Information Systems
- Microsoft Azure Cloud Environment
- Active Directory
- Microsoft 365
- GitHub Enterprise Repositories
- AI Models and Model Weights
- AI Training Datasets
- Security Information and Event Management (SIEM)
- VPN Infrastructure
- Internal Web Applications
- Authentication Systems
- Intellectual Property

---

## Out-of-Scope Assets

The following assets are excluded from this assessment:

- Customer-owned systems
- Third-party managed infrastructure
- Personal employee devices (BYOD)
- External partner networks
- Physical security controls
- Production systems requiring uninterrupted availability

---

## Audit Methodology

The assessment follows a structured security auditing methodology consisting of:

1. Documentation Review
2. Stakeholder Interviews
3. Asset Identification
4. Security Configuration Review
5. Risk Assessment
6. Compliance Assessment
7. Findings Validation
8. Final Reporting

---

## Assessment Standards

This assessment is conducted in accordance with the following industry standards and best practices:

- NIST Cybersecurity Framework (CSF) 2.0
- NIST SP 800-53 Rev. 5
- ISO/IEC 27001:2022
- CIS Critical Security Controls v8

---

## Audit Deliverables

The final deliverables include:

- Executive Summary
- Asset Inventory
- Risk Assessment Report
- Security Findings
- Compliance Assessment
- Risk Matrix
- Security Improvement Recommendations
- Final Audit Report
